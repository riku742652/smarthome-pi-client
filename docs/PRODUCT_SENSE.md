# Product Sense - Smarthome Pi Client

## プロダクトの目標

SwitchBot CO2センサーのデータを手動操作なく自動収集し、smarthome ダッシュボードに継続的に供給する。

## ユーザーストーリー

- **As a** スマートホームオーナー
- **I want** Raspberry Pi が自動でセンサーデータを収集・送信してほしい
- **So that** ダッシュボードで温度・湿度・CO2 濃度をリアルタイムに確認できる

## 優先順位

1. **信頼性**: センサーデータが欠損なく送信されること
2. **自動化**: 手動介入なく継続的に動作すること
3. **観測可能性**: ログで動作状況を把握できること
